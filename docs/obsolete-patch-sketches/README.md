# Obsolete Patch Sketches

These files are retained only as historical notes from the early manual recovery phase.

They are not valid/recommended inputs for `patch(1)` and should not be used by the build script.

The current reproducible build path uses:

    scripts/apply-linux-wsl-patches.py

The earlier `.patch` files were intentionally superseded after clean reproduction testing showed they were brittle/documentation-shaped rather than reliable unified patches.
